2. Codi Font

Implementació C (Motor d'Inferència)

  • main.c C

    Punt d'entrada, càrrega de pesos i pipeline d'inferència complet.

  • sorteny_layers.c C

    Implementació de capes: Conv2D, GDN, Dense, ReLU, Spectral Transform.

  • sorteny_model.c C

    Càrrega dinàmica de pesos des de fitxers TSV/binaris.

  • sorteny_model.h C

    Estructures de dades: LayerWeights, SortenyModel.

  • io_helpers.c C

    Funcions E/S per imatges RAW (BSQ uint16 → float32).

  • io_helpers.h C

    Declaracions de les funcions d'entrada/sortida.

Codi Python

  • src/python/ Python

    Scripts de validació, benchmarks i utilitats (24 arxius organitzats).

  • scripts/ Bash

    Scripts d'automatització per benchmarks i desplegament a Raspberry Pi.
